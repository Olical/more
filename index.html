more

A framework for LESS

Incorperating normalisation, a customisable grid system, vendor prefixes and other useful mixins such as #more .circle(10px).

You will probably learn more from checking out the source but this page should get you going. More is a framework written in LESS, an intermediate language for CSS, that can also be used with any normal CSS based project. This is because precompiled versions are provided in the repository. It allows you to forget about resets, normalisation, grid systems and vendor prefixes by handling it all for you.
